## 1. 产品概述
低空充电与电池换电服务 Web 平台，面向无人机运营商和补能站点管理员，提供一站式补能预约与管理服务。解决无人机低空作业的续航痛点，实现补能资源的高效调度与运营管理。

- 目标用户：无人机运营商、补能站点管理员、平台运营方
- 核心价值：提升补能效率、优化资源配置、降低运营成本

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 无人机运营商 | 手机号注册 | 站点搜索、预约补能、查看充电进度、费用结算、发票申请、会员管理 |
| 站点管理员 | 平台分配 | 设备状态监控、预约核销、电池库存管理、异常工单处理、经营数据查看 |
| 平台运营 | 后台账号 | 全平台数据统计、站点管理、会员体系配置、报表生成 |

### 2.2 功能模块
1. **站点地图**：站点搜索、接口类型筛选、地图展示、站点详情
2. **设备状态**：充电桩/换电柜实时状态、故障告警、设备监控
3. **预约排队**：预约时段选择、队列管理、到站核销、超时提醒
4. **电池库存**：可用电池查询、库存盘点、电池编号绑定、出入库记录
5. **充电记录**：充电进度查看、历史订单、费用明细
6. **会员账户**：押金管理、会员折扣、账户余额、消费记录
7. **异常工单**：设备故障上报、工单处理、超时占位提醒
8. **经营报表**：运营收入统计、客户评价、数据分析

## 3. 核心流程

### 3.1 预约补能流程
用户打开站点地图 → 搜索筛选目标站点 → 查看站点详情和可用时段 → 选择服务类型和时段 → 提交预约 → 支付押金/费用 → 生成预约码 → 到站扫码核销 → 开始充电/换电 → 服务完成 → 费用结算 → 评价服务

```mermaid

    A["打开站点地图"] --> B["搜索筛选站点"]
    B --> C["查看站点详情"]
    C --> D["选择服务类型和时段"]
    D --> E["提交预约"]
    E --> F["支付押金/费用"]
    F --> G["生成预约码"]
    G --> H["到站扫码核销"]
    H --> I["开始充电/换电"]
    I --> J["服务完成"]
    J --> K["费用结算"]
    K --> L["评价服务"]
```

### 3.2 电池换电流程
用户到站 → 扫码核销 → 选择换电服务 → 放入旧电池 → 系统识别电池编号 → 取出新电池 → 绑定新电池编号 → 完成换电 → 费用自动结算

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：科技蓝 (#165DFF) 搭配能源绿 (#00B42A)，体现科技感与清洁能源属性
- **辅助色**：警示橙 (#FF7D00) 用于异常状态，中性灰用于文本和背景
- **按钮风格**：圆角 8px，主按钮使用渐变蓝色，悬停时有微上浮动效
- **字体**：标题使用 Noto Sans SC 600，正文使用 Noto Sans SC 400，数字使用等宽字体
- **布局风格**：左侧导航栏 + 顶部状态栏 + 主内容区的经典后台布局，卡片式内容展示
- **图标风格**：线性图标，统一 24px 尺寸，选中状态使用主色填充

### 4.3 响应式
- 采用桌面优先设计，主内容区最小宽度 1200px
- 平板端自适应调整侧边栏宽度，表格支持横向滚动
- 移动端折叠侧边栏为汉堡菜单，优化表单输入体验

### 4.4 交互动效
- 页面切换使用淡入淡出过渡，时长 300ms
- 数据卡片悬停时有轻微上浮和阴影加深效果
- 进度条使用平滑动画更新
- 通知消息从右上角滑入，自动消失
- 地图标记点点击时有弹跳动效
